# Nima Karimian Centered Academic Website

This version fixes the left-heavy design. It has:
- Centered hero and circular profile photo
- Centered section headings
- Centered teaching/projects/students/service cards
- Publication filtering
- Dark/light mode
- Social icons: Email, Google Scholar, GitHub, LinkedIn, X/Twitter
- Visitor map placeholder

Upload all extracted files to the root of your GitHub repository.

To replace the photo:
1. Add your photo as `assets/profile.jpg`
2. In `index.html`, replace `assets/profile-placeholder.svg` with `assets/profile.jpg`

To update Scholar metrics, edit `scholar_data.json`.
