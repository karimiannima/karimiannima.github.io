# Nima Karimian Academic Website

Personal academic website of **Nima Karimian, Ph.D.**
Assistant Professor, Bellini College of Artificial Intelligence, Cybersecurity and Computing,
University of South Florida.

🌐 **Website:** https://karimiannima.github.io/

---

## Features

- 🎓 Modern academic homepage
- 📄 Publications with filtering
- 🔬 Research projects
- 👨‍🎓 Students
- 📚 Teaching
- 💼 Professional service
- 🏆 Awards
- 📈 Google Scholar metrics
- 🌍 Live visitor map
- 🌙 Dark / Light mode
- 📱 Fully responsive design

---

## Update Profile Photo

Replace

```
assets/profile-photo.jpg
```

with your new image.

---

## Update CV

Replace

```
assets/Curriculum_Vitae-Nima.pdf
```

with the latest version.

The homepage button will automatically open the updated CV.

---

## Update Google Scholar Metrics

Edit

```
scholar_data.json
```

Example:

```json
{
  "citations": 1250,
  "h_index": 21,
  "i10_index": 32
}
```

---

## Update Publications

Edit the **Publications** section in

```
index.html
```

Each publication supports filtering by:

- Year
- Conference / Journal
- Topic

---

## Update Visitor Map

Create a free visitor map from one of:

- ClustrMaps
- FlagCounter
- RevolverMaps

and replace the placeholder in the **Visitors** section with the generated embed code.

---

## Local Development

Simply open

```
index.html
```

in your browser.

No build process or dependencies are required.

---

## Deployment

The website is automatically deployed using **GitHub Pages** via GitHub Actions.

After pushing changes to the `main` branch:

```
git add .
git commit -m "Update website"
git push origin main
```

GitHub Actions will automatically publish the latest version.
